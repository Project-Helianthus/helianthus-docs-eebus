# AGENTS

This repository is part of the Helianthus multi-protocol HVAC gateway platform.

Use the workspace-root `AGENTS.md` for orchestration, review, doc-gate, and
transport-gate rules.

Repository-specific rules:

1. Keep this repo eeBUS-native. Cross-protocol platform contracts belong in
   `helianthus-docs-ebus/docs/platform/`.
2. Do not publish private identifiers, credentials, account data, household
   schedules, private network addresses, or full device fingerprints.
3. Do not publish restricted-source material or paraphrases of restricted
   material.
4. Every protocol claim must cite publishable evidence or remain a hypothesis.
5. Operator notes are context only and never establish a protocol fact.
